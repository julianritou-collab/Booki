Ce projet représente le prototype de la page d'accueil de l'agence de voyage Booki.

Le projet contient :
- un fichier index.html correspondant à la page d'accueil ;
- un dossier CSS contenant les deux feuilles de style utilisées (style.css et cards.css):
    style.css : feuille de style principale,
    cards.css : feuille de style dédiée aux sections utilisant des cartes;
- un dossier images contenant toutes les images utilisées sur le site.

Les fichiers CSS sont organisés selon une approche desktop first, puis adaptés aux tablettes et enfin aux mobiles.
